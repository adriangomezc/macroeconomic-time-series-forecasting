# Macroeconomic Time Series Forecasting: Spain–EU Imports

This repository develops a forecasting framework for monthly Spanish imports from European countries using classical and modern time series methods in R.

The objective is to analyse long-term trade dynamics, identify structural shocks and seasonal patterns, and build robust forecasting models for medium-term import projections.

The dataset contains monthly import volumes (kilotons) from January 2000 to May 2025 obtained from Eurostat.

---

# Business context

International trade flows are strongly affected by:

- Economic cycles
- Financial crises
- Supply chain disruptions
- Calendar effects
- Seasonal industrial activity

Understanding these dynamics is essential for:

- Supply chain planning
- Macroeconomic monitoring
- Trade forecasting
- Industrial logistics
- Economic policy analysis

This project explores how statistical forecasting models can capture these dynamics under real-world macroeconomic conditions.
